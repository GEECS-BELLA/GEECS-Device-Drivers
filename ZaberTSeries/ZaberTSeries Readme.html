Zaber Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Zaber Technologies
Supported Language: LabVIEW
Supported Models: T-CD, T-JOY, T-LA, T-LHM, T-LLS, T-LS, T-LSM, T-LSQ, T-LSR, T-MM, T-NA, T-NM, T-OMG, T-RS, T-RSW (T Series), A-BAR, A-BLQ, A-LAR, A-LSQ, A-LST, A-LAR, A-MCA, A-MCB2 (A-Series), X-BLQ, X-DMQ, X-FWR, X-GLP, X-GSM, X-JOY3, X-LHM, X-LRM, X-LRQ, X-LSM, X-LSQ, X-MCB1, X-MCB2, X-NMS, X-RSB, X-RST, X-RSW, X-VSR (X-Series)
Models Tested: T-RS, T-LSM, T-JOY
Interface: Serial
Firmware Revisions Tested: 5.11, 5.12

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.1
Original Release Date: March 20, 2009
Current Revision Date: April 18, 2017
Release Date: June 26, 2017

For more documentation and a tutorial, go to the Zaber Technologies web site.

VISA 5.0 or later
