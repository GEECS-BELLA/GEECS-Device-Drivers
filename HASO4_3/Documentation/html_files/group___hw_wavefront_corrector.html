Interfacing the Wavefront Corrector

Detailed Description

All Editions.

WavefrontCorrector parametrization and control.

A Wavefront corrector is a generic device allowing Wavefront modulation. It can be a deformable mirror, a SLM ... Similarly to the camera device, the wavefront corrector device management is split in two classes :

  • The WavefrontCorrectorSet module exposes the wavefront corrector specifications (size, serial number, list of available parameters, parameters types ...).
  • The WavefrontCorrector module is the interface with the real device.

For example, if you want to set parameter P:

  • check its existence, type and characteristics using the WavefrontCorrectorSet module,
  • use the appropriate method of the WavefrontCorrector module to truly modify P value.
