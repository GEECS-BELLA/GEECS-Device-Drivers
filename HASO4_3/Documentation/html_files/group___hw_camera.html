Interfacing the Camera

Detailed Description

All Editions.

Camera parametrization and image acquisition.

The camera device management is split in two classes :

  • The CameraSet module exposes the camera specifications (size, serial number, list of available parameters, parameters types ...).
  • The Camera module is the interface with the real device.

For example, if you want to set parameter P:

  • check its existence, type and characteristics using the CameraSet module,
  • use the appropriate method of the Camera module to truly modify P value.
