Advanced computation from an image
Computing and manipulating the wavefront slopes

Detailed Description

Expert.

The HasoEngine manages all the advanced parameters involved in the HasoSlopes computation from an image acquired with a Haso sensor, as well as the computation itself.

It proposes two differents algorithms to compute slopes, see E_SPOTDETECT_T documentation.

The following parameters are exposed to fine tune the HasoSlopes computation results :

  • The shift (expressed in sub-pupils count, in x and y directions)
  • The coordinate of the reference sub-pupil used to initiate the spots detection.
  • The sensitivity of spots detection between 0 and 1 (in case summed images are used) which modulates noise management when computing slopes on summed images.
    • 0 : Release constraint on noise to detect as much sub-pupils as possible.
    • 1 : Do not take into account the noise reduction caused by summing and keep restrictive parameters.
