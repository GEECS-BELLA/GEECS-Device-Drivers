Advanced phase reconstruction
Computing and manipulating the wavefront phase

Detailed Description

Expert.

Fonctions dedicated to phase reconstruction.
These functions reconstruct phase from processed slopes (see Advanced slopes processing) in a HasoData object using parameters and filter stored in a Imop_ComputePhaseSet container.
Depending on the reconstruction type, the parameters stored in the Imop_ComputePhaseSet can be filtering settings and / or reconstruction algorithms parameters.
