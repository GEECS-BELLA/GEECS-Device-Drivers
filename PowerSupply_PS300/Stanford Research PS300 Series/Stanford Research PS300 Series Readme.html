Stanford Research PS300 Series Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Supported Language(s): LabVIEW
Manufacturer: Stanford Research
Interface(s): GPIB

Supported Model(s): PS310, PS325, PS350, PS355, PS365, PS370, PS375
Model(s) Tested: PS355
Firmware Revision(s) Tested: None

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 2.1.1
Original Release Date: 01/01/1997
Current Revision Date: 06/12/2015

2. Required Software

Some software components need to be installed before using this instrument driver. Minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 5.0 or later

For more details, refer to the LabVIEW Help under Help » Search the LabVIEW Help.

3. Known Issues

If having problems setting output voltage, voltage limit, or current limit, try negating your voltage or current value.

To report issues or provide feedback about this instrument driver, please contact technical support.
