Read From Port

The "Read from Port" function is used to read the responses from the VXM.

To call this function do the following:

  • 1.) In your code window click "Select a VI"
  • 2.) Choose the VxmDriverLV8.vi (found in the driver folder)
  • 3.) Place this vi in your code window

  • 4.) Wire a boolean true to "Read from Port" connection
  • 5.) Wire a string indicator to hold the characters read from the VXM from the "Port Chars" connection
