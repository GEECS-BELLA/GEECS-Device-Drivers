GHV232 Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Glassman High Voltage, Inc.
Supported Language(s): LabVIEW
Supported Model(s): All ACI-RS-0x interfaces for Glassman High Voltage Power Supplies
Model(s) Tested: ACI-RS-09
Interface(s): RS232, USB (only with virtual com port drivers)
Firmware Revision(s) Tested: 03

Certified: No
NI Supported: No
Source Code Available: Yes

Driver Revision: 1.0
Original Release Date: 05/24/2006
Current Revision Date: 06/02/2008
